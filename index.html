APIStrat Austin 2015

The API Conversation Continues...

WHEN November 18-20, 2015
WHERE Austin, TX

About

API Strategy & Practice Conference, now known as APIStrat, is a conference focused on the API economy. The sixth edition of the conference is coming to Austin TX, looking to bring together API leaders, all the way to the API curious, and discuss opportunities and challenges from the space.

APIStrat brings together API providers, service providers, and integrators to discuss the what they see each day on the ground. The event is not just for developers, and IT, it also mean for business users, and anyone who is looking to better understand how APIs are impacting the world around them.

Thanks for visiting, stay tuned for more information, and we look forward to seeing you Austin this fall.
