APIStrat Austin 2015

The API Conversation Continues...

WHEN November 18-20, 2015
WHERE Austin, TX

About

API Strategy & Practice Conference, also known as APIStrat is a conference focused on the technology, business, political aspects of the API economy. The sixth edition of the conference is coming to Austin TX, and is looking to bring together API leaders, to the API curious, to discuss the biggest challenges in the space.

APIStrat brings together API providers, and service providers, to discuss the opportunities, and challenges in the API space. The event is not just for developers, and IT, it also mean for business users, and anyone who is looking to better understand how APIs are impacting their world.

Thanks for visiting, stay tuned for more information, and we look forward to seeing you Austin this fall.
